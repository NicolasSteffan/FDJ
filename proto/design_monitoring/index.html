💡 Guide d'utilisation

  • Cliquez sur une carte pour ouvrir le prototype dans un nouvel onglet
  • Chaque prototype est entièrement fonctionnel avec ses interactions spécifiques
  • Testez la responsivité en redimensionnant la fenêtre de votre navigateur
  • Les prototypes mobiles sont optimisés pour la consultation sur smartphone/tablette
  • Explorez les différentes approches UX pour identifier celle qui convient le mieux à vos besoins
  • Tous les prototypes utilisent les mêmes données de démonstration pour faciliter la comparaison
